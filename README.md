# Client Tracker

## Project Overview
The Client Tracker project is designed to streamline the process of managing immigration documentation for clients. This application helps users track various types of immigration documents, ensuring that all necessary paperwork is collected and managed effectively.

## Setup Instructions
To set up and run the Client Tracker application on your local machine, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/jccuenca/Client-Tracker.git
   cd Client-Tracker
   ```

2. **Install dependencies**:
   ```bash
   npm install
   ```

3. **Run the application**:
   ```bash
   npm start
   ```
   The application will be running on `http://localhost:3000`.
   
## Document Types Tracked
The following immigration documents are tracked by the Client Tracker application:

1. **Passport** - A government-issued identification document that allows for international travel.
2. **Visa** - An endorsement placed within a passport that grants permission to enter, stay in, or leave a country.
3. **Green Card** - A document that provides proof of a person's permanent residency in the United States.
4. **Work Permit** - An official authorization allowing a foreign national to work in a specific country.
5. **I-20 Form** - A Certificate of Eligibility for Nonimmigrant Student Status used for international students in the U.S.
6. **I-94 Form** - A document issued by U.S. Customs and Border Protection (CBP) that shows your arrival and departure from the U.S.
7. **Naturalization Certificate** - A document that proves a person has become a U.S. citizen.
